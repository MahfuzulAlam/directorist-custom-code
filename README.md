# Import Business Hour
This extension will allow users to import business hours while importing listings using the Directorist default import feature/option. This extension only supports the business hour format from google business hours.

# Import Format
```
{"Monday": "12pm-9:30pm", "Tuesday": "12pm-9:30pm", "Wednesday": "12pm-9:30pm", "Thursday": "12pm-9:30pm", "Friday": "12pm-9:30pm", "Saturday": "Closed", "Sunday": "Closed"}
```

# Import Rule
- Please make sure to put all hours in am/pm format. Do not leave like this - "12-9:30pm". It does not make sense.
- Please make this this option is set to 12 hours format - https://prnt.sc/uskESe0Qi4rb
